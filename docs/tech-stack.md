# tech stack

1. solana for the underlying entire infra
2. rust and anchor framework for writing the solana program
3. risc0 for ZKPs (\*)
4. IPFS for p2p file hosting
5. solana attestation for associating and validating offchain data (document commitment) with onchain accounts
6. surfpool and lightSVM for forking and simulating solana mainnet locally on laptop
7. bun runtime for writing unit tests for the program in typescript
